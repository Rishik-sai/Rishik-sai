## 🚀 Projects

### 🌾 [AgriMitra AI | Multi Agent Agricultural Intelligence Platform](https://github.com/Rishik-sai/agrimitraai)
> *LangGraph · Corrective RAG · FAISS · CrossEncoder · Groq Vision · FastAPI · React · Redis · Prometheus*

**[🔴 Live Demo](https://agrimitraai-6cna.vercel.app/) · [📖 API Docs](https://agrimitra-backend1.onrender.com/docs) · [💻 GitHub](https://github.com/Rishik-sai/agrimitraai)**

Engineered a **self correcting Corrective RAG (CRAG) pipeline** as a LangGraph `StateGraph` with conditional routing across retrieve, evaluate, web search fallback, synthesize, and reflect nodes. Deployed **five specialized agentic pipelines** (Crop Advisor, Market Analyst, Schemes Expert, Weather Analyst, Leaf Scanner) over async **SSE streaming** on FastAPI.

* 🔬 Two stage retrieval: **FAISS** vector search (`paraphrase-multilingual-MiniLM-L12-v2`) + **CrossEncoder** reranking
* 📊 RAGAS evaluated: **0.80 Faithfulness · 0.90 Answer Relevance · 0.70 Context Recall** on a 30 pair evaluation set
* 👁️ **Groq Vision** (Llama 4 Scout 17B) for multimodal leaf disease diagnosis under 3s
* 🌐 LLM translation pipeline across **11 Indian languages**
* ☁️ Deployed on **Vercel + Render** with **GitHub Actions CI**, **Redis** caching, and **Prometheus** observability

### 🌐 [Universal Browser Assistant | Agentic Browser AI Extension](https://github.com/Rishik-sai/universal-browser-assistant)
> *Chrome Manifest V3 · LangChain · FastAPI · MongoDB · Redis · Tavily · Web Speech API*

**[🟢 Live API](https://uba-backend.onrender.com/docs) · [💻 GitHub](https://github.com/Rishik-sai/universal-browser-assistant)**

Built a **Chrome Manifest V3 extension** injecting a **Shadow DOM isolated AI chat sidebar** into any webpage, grounding LLM responses in up to 3,000 characters of live page context via a **LangChain powered FastAPI backend** with async **MongoDB (Motor)** persistence and **JWT authentication**, deployed live on Render.

* 🔍 Integrated **Tavily Search API** for real time highlight to ask web search
* ⚡ **Redis backed session cache** and rate limited authentication endpoints
* 🌐 In place page translation into **9 Indian regional languages** via Groq LLMs with one click DOM restoration
* 🎙️ Full **voice to inference to audio pipeline** using the **Web Speech API**
